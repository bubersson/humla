Highlighting Syntax

  • Example Javascript code
    •                         // initialize the array
                        var a = ["one", "two", "three", "four"];
                        
                        // example loop
                        for (var i = 0; i < a.length; i++) {
                            if (a[i] == "two")
                                break;
                        }
    • line 5: loop through elements in the array
      • array has a property length that contains a number of elements in the array (cf. line 2)
    • line 7: break when the element two was found
      • the loop will stop, the execution will continue with code after the loop

Google Drawing

  • Draw figures in Google, use them in Humla
    • Figures are dynamically linked from Google Drawing
    • Any changes in the source drawing immediatally appear in Humla presentation

Github Integration

  • Submit your code to github, link it in Humla
    • Code can be dynamically linked from Github
    • You can link it with a particular commit to preserve a slide layout
